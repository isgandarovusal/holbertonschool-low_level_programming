# Variadic Functions Project

This project implements variadic functions in C, which are functions that can accept a variable number of arguments.

## Files

1. **variadic_functions.h** - Header file containing all function prototypes
2. **0-sum_them_all.c** - Function that returns the sum of all its parameters
3. **1-print_numbers.c** - Function that prints numbers with a separator
4. **2-print_strings.c** - Function that prints strings with a separator
5. **3-print_all.c** - Function that prints anything based on format string
